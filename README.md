## Сбор данных

Для сбора данных использовался официальный VK API. Собраны поля:
- ID поста
- Дата публикации
- Количество лайков
- Текст поста

День недели рассчитан программно на основе даты (`1 = понедельник, ..., 7 = воскресенье`).

Основной код приложен в файле: [main.py](main.py)
